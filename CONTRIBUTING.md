# Contributing

Thanks for checking out `hermes-agent-3ds`.

## Good first contributions

- bug fixes
- UI polish
- docs cleanup
- setup improvements
- small quality-of-life features

## Before you start

- open an issue if the change is big
- keep PRs focused
- prefer simple solutions over clever ones
- keep user-facing copy short and human

## Style

This project should feel friendly, lightweight, and easy to use.

That means:
- avoid walls of text
- avoid overly technical wording in front-facing docs
- keep screens readable on real 3DS hardware
- respect the project's retro Hermes visual direction

## Pull requests

Please include:
- what changed
- why it changed
- how you tested it
- screenshots if the UI changed

## Be kind

Assume good intent, give useful feedback, and keep things constructive.
